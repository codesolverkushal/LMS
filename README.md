# LMS Frontend

### Setup Information

1. Clone the Project
...

    git clone https://github.com/codesolverkushal/LMS.git
  
...
2. Move into directory
...

    cd lms-frontend (if you already in it, just ignore)
  
...
3. install dependencies
...

    npm start
  
...
4. Run the server
...

    npm run dev
  
...
